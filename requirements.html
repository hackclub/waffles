What You Need to Build

Check this page for all the requirements for your submission!

01

The Basics

Two Pages
Create two different HTML pages with real content. No placeholders, no lorem ipsum - actual stuff :D
Your Work
Make sure the project is your own! Do not copy and paste code from the internet.
Make It Yours
Add something that makes the site fun and different so it does not feel generic. You should give it personality.
Features
Use at least five HTML and CSS features on each page; show what you know.
Static Only
The site should be static, which means just HTML, CSS, and JavaScript. No frameworks needed.
Content
Put some text, colors, and at least one image on each page. Make sure it looks complete!
Fresh Build
The project must be created recently (1 month maximum). Old projects will not count.
02

JavaScript Functions

Your site needs three different ways for a user to change the content in a meaningful way.

Use One of These JS Features
At least one of your functions must use one of the following:
Pick at least one:
  • fetch - a special way to get information from other websites!
  • setTimeout - delay something from happening
  • setInterval - repeat something on a timer
Function Type 1: Change Something
Your function should change or update something on the page.
Examples:
  • Swap an image when it is clicked
  • Hide or show part of the page
  • Change the style of an element
Function Type 2: React to User
Your function should react to something the user does.
Examples:
  • Run when someone clicks a button
  • Check a form before it is submitted
  • Update text live while the user is typing
Function Type 3: Use Core JavaScript
Your function should use a core JavaScript idea.
Pick from these:
  • Event listeners
  • If statements
  • Updating the DOM
  • Changing styles with JavaScript
  • Variables and functions
03

Important Notes

Each function should clearly affect the page so the user sees what happens.

Use simple, clear names for your functions.

Do not copy from websites.

AI is not allowed. This includes using AI to clean up your work, generate filler text, create images, or write any part of your code.
