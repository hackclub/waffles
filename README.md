## Running Locally

1. **Clone the repo**
   ```
   git clone https://github.com/hackclub/waffles.git
   ```

2. **Navigate into the directory**
   ```
   cd waffles
   ```

3. **Install dependencies**
   ```
   npm install
   ```

4. **Start the dev server**
   ```
   npm run dev
   ```

Then open the local URL shown in your terminal!
